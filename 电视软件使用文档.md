# 小米电视通过甲壳虫ADB软件安装第三方app

1. 手机通过TiviStore商店下载 甲壳虫ADB软件
2. 通过相关步骤开启电视ADB功能
3. 输入电视的IP地址，连接 （点击放大镜搜索电视IP地址或者通过电视无线/有线网络查看IP）
4. 安装apk软件
